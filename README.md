# 📁 Файловый менеджер (Лабораторная работа №4)

## Описание
Консольный файловый менеджер для работы с файлами и каталогами. 
Разработан в рамках учебной лабораторной работы.

## 🚀 Функциональные возможности
- **Каталоги:** создание, просмотр, перемещение, переименование, копирование, удаление, атрибуты, поиск, права.
- **Файлы:** создание, просмотр, редактирование, перемещение, переименование, копирование, удаление, атрибуты, поиск, права.

## 🛠 Технологии
- **Язык:** Python 3.x
- **Библиотеки:** os, shutil, pathlib, stat (стандартные)
