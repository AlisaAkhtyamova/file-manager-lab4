# Файловый менеджер (Лабораторная работа №4)

## Описание
Программа для управления файлами и каталогами (создание, удаление, копирование, права доступа).

## Установка
1. Установите Python 3.x
2. Клонируйте репозиторий
3. Запустите: `python main.py`

## Функционал
- Работа с файлами и папками
- Поиск по маске
- Изменение прав доступа
